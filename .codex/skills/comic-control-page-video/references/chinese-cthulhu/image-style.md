# 中式克苏鲁图像风格规范

用于中式克苏鲁怪谈模式的参考图包、导演统筹和 9:16 单页剧情图 prompt 管理。
目标是统一视觉气味：中式民俗空间、潮湿纸质感、克制的不可名状恐怖。

## 文字 Prompt 是硬约束

图像 prompt 必须写清：

- 这是原创中式克苏鲁 / 民俗怪谈视觉，不是历史复原图。
- 不出现真实历史人物、真实朝代符号堆砌、宫廷权谋或战争场面。
- 默认不走欧美哥特城堡、赛博邪神、游戏 boss、克苏鲁海报、廉价触手怪路线；但用户明确要求“游戏怪物、触手、古神、邪神 boss”时允许强化怪物主体。
- 恐怖优先来自空间、仪式、材质、凝视和认知错位；当用户要求怪物化时，可以给出怪物全身、局部特写或 boss 感，但要让它长在中式民俗语境里。

## 默认风格 Prompt

构建 `reference_board_prompts.md`、`director_visual_plan.md` 或
`vertical_page_prompts.md` 时，默认并入这段风格 prompt。用户另给风格 prompt
时，以用户版本为准，但仍保留“不写历史”和参考图 manifest 规则。

```text
原创中式克苏鲁民俗怪谈漫画风格。

整体视觉是潮湿、压抑、古老、带有仪式感的中式恐怖世界，不是历史复原图。场景根植于虚构的中国乡土空间：祠堂、古井、祖屋、河湾、旧庙、族谱库房、坟山石阶、潮湿戏台、荒村夜路。

画面材质以湿纸、霉斑、朱砂、香灰、黑水、盐霜、暗红木漆、青石、锈铁、油灯烟、旧神像裂纹为主。色彩低饱和，墨黑、灰绿、脏白、旧木褐、暗红、微弱冷青；朱砂红只作为危险信号和仪式痕迹。

允许出现强视觉冲击的古神、邪神 boss、触手巨物、巨眼、黑水肢体、肉质化神像、从井壁或祠堂梁柱里长出的不可名状结构。怪物必须中式民俗化：纹理来自朱砂符、木雕、牌位漆面、井绳、水草、香灰、霉纸、青石裂缝和旧庙壁画，而不是默认西式章鱼海怪。

触手不是无意义堆满画面，必须从井水、族谱页缝、神像袖口、祠堂梁柱、黑潮、牌位背面或墙体霉斑里伸出。古神不需要完整解释，可以只表现为一只眼、一截肢体、一片倒悬天空、一尊正在变肉的神像，或者半藏在黑水与烟雾后的巨大轮廓。

画面气质要像严肃漫画分镜和概念设计结合：清晰构图、强光影、可读角色、
电影感但不要海报化。恐怖来自空间、仪式、材质、凝视和认知污染，同时保留
怪物压迫感。默认是一 beat 一张 9:16 竖版单页图，不画漫画格，不输出控制页。

降低 AI 生图噪点：中式克苏鲁的恐怖不靠满屏纹理和伪细节。构图优先，先保证人物轮廓、空间关系、分镜读序和关键物件清楚。减少无关人物、背景杂物、旧纸碎字、密集符号、衣物花纹、随机霉斑、碎触手和过锐化边缘。只在井口、牌位、红线、神像眼洞、黑水反光等关键叙事点保留少量可读纹理，其余区域使用干净线条、低纹理材质、柔和光影和低饱和色块。
```

配套负面 prompt：

```text
不要真实历史人物，不要真实朝代复原，不要宫廷权谋，不要战争史诗。
不要现代 UI、社交媒体界面、血条、技能面板、装备栏、网游截图感。
不要欧美哥特城堡，不要赛博霓虹，不要科幻机械邪神。
不要廉价触手海报，不要纯章鱼海怪，不要过度血浆，不要萌系、Q版、轻小说风。
不要把怪物画成普通游戏怪兽皮肤，必须保留中式民俗材质和仪式污染感。
不要超细节、战损、极致纹理、过度锐化、密集小纹理、随机噪点、满屏碎片、无数裂纹、无意义背景杂物、无叙事作用的人群和摆件。
```

## 图片参考文件夹

本项目默认使用这个参考图片目录：

```text
local-reference-images/chinese-cthulhu
```

repo 相对路径：

```text
中式克苏鲁参考图片/
```

只有用户另给参考目录时，才覆盖默认目录。若需要新增临时参考图，可以放在项目输入目录：

```text
project_input/style_refs/chinese-cthulhu/
```

建议 8 到 20 张，不要超过 50 张。按用途命名更好：

```text
space-ancestral-hall-01.png
space-old-well-01.png
material-mold-paper-01.png
lighting-oil-lamp-01.png
monster-partial-eye-01.png
```

主控需要生成：

```text
00_brief/style_reference_manifest.json
```

记录每张参考图的路径、sha256、用途、是否实际传给图像工具。如果工具不能传图，必须写 `supports_image_reference=false`，并把图片气质转成文字约束。

## 视觉关键词

- 空间：祠堂、古井、河湾、祖屋、庙门、义庄、潮湿库房、旧戏台、荒村小路、坟山石阶。
- 材质：湿纸、霉斑、朱砂、香灰、黑水、盐霜、暗红木漆、破旧蓝布、青石、锈铁、油灯烟。
- 光线：低照度、油灯边缘光、雾气散射、井底冷光、暗水反光、纸窗透光。
- 怪异：无眼神像、族谱空白页渗字、井底星光、牌位背面长出纹路、影子错位、墙面像皮肤一样鼓起。
- 色彩：低饱和墨黑、灰绿、暗红、脏白、旧木褐、微弱冷青；朱砂红只作局部危险信号。

## 怪物 / 触手 / 古神

用户明确要求时可以使用：

- 游戏怪物、邪神 boss、古神显形、触手巨物、巨眼、黑水肢体、长在祠堂梁柱或井壁上的肉质结构。
- 设计方向要中式化：纹理来自朱砂符、木雕、牌位漆面、井绳、青石、水草、香灰、霉纸、旧神像裂纹，而不是默认西式章鱼海怪。
- 可以做强视觉冲击的大怪物，但不要加入血条、技能 UI、装备面板、科幻机械、霓虹赛博、欧美城堡和纯网游截图感。
- 触手要有功能和来源：从井水、族谱页缝、神像袖口、祠堂梁柱、黑潮或牌位背面伸出；不要无意义堆满画面。
- 古神不需要完整解释。可表现为“局部显形”：一只眼、一截肢体、一片倒悬天空、一尊正在变肉的神像。
- 怪物复杂度要可控：用清晰大轮廓、少量标志性纹理和明确负空间制造压迫感，不用满屏小触手、碎肉纹、随机符号或密集噪点制造“复杂感”。

## 参考图包

参考图包必须服务 9:16 单页剧情图的一致性。中式克苏鲁模式下至少包含：

- `character_prop_reference.png`：主角三视图、5 个表情、关键配角、关键道具、
  污染状态变化。
- `location_plan_reference.png`：村落总平面、祖屋 / 祠堂 / 古井 / 小庙 / 河湾 /
  城市管道的位置关系，以及人物动线图。
- `shot_style_reference.png`：灯光、色彩、材质、镜头距离、构图和怪异揭示节奏。
- 可选关键物件：只有出场超过 3 次或直接影响剧情 / 仪式 / 悬念的物件才纳入，
  例如族谱、井绳、油灯、红线、黄纸、铜钱、黑水碗、无名牌位；不要为了填满
  参考图强塞普通民俗道具。

## 9:16 单页 Prompt 补充

每张 9:16 单页 prompt 应包含：

- 当前 beat 的恐怖阶段：日常异常 / 规则提示 / 接近禁忌 / 认知污染 /
  不可名状局部 / 代价结尾。
- 当前 beat 的主视觉物件：井、族谱、牌位、门缝黑水、空白照片、油灯、
  墙面眼状霉斑等。
- 导演指令：景别、视角、运镜意图、人物站位、道具位置、场景平面关系和灯光。
- 气氛材质：潮湿、低饱和、纸张发霉、暗水反光、朱砂局部。
- 禁止项：现代 UI、游戏界面、欧美哥特、科幻机械、血浆堆砌、廉价触手海报、
  漫画格、边框、对白气泡、把参考图拼进剧情图。若用户要求游戏怪物或古神，
  可以出现怪物全身设定图，但必须中式民俗化。

## 图像自检

- 是否像中式民俗怪谈，而不是西式克苏鲁海报。
- 是否过度依赖触手、巨眼和怪物正脸；用户要求强怪物时可以放大主体，但仍要有中式空间和材质支撑。
- 是否出现伪细节、过度锐化、高频纹理、无关杂物或满屏碎触手；如果出现，应删减背景和材质，保留干净线条、低纹理与柔和光影。
- 是否是一张完整 9:16 竖版剧情图，而不是控制页、漫画格或参考图拼贴。
- 是否延续同一参考图包中的人物、场景总平面、灯光和动线。
- 是否少特写、多场面调度，且能读出空间关系和人物位置。
